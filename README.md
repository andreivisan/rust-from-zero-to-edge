# RUST - FROM ZERO TO EDGE

## Introduction

The purpose of this repository is to get every developer (with previous software
development experience) from zero to:

1. Great knowledge of Rust fundamentals
2. Great knowledge of Rust intermediate and advanced idiomatic coding
3. Solid understanding of data structures in Rust
4. Experience solving LeetCode interview problems in Rust
5. Great low-level/systems programming in Rust
6. Good understanding of AI, ML, and Edge AI/ML
7. Implementing Edge AI on different hardware architectures

The way this will be achieved is through the following:

- Building small projects
- Ending with a bigger project that combines the learned concepts
