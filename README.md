# RUST - FROM ZERO TO EDGE

## Introduction

The purpose of this application is to get every developer (with previous software
development experience) from zero to:

    1. Great knowledge of Rust fundamentals
    2. Great knowledge of Rust intermediate and advanced idiomatic coding
    3. Fully experienced in understand data structures in Rust
    4. Fully experienced in solving LeetCode interview problems in Rust
    5. Great low level/systems programming in Rust
    6. Good understand of AI ML and Edge AI ML
    7. Implement Edge AI on different hardware architectures

The way this will be achieved is through the following:

    - Building small projects
    - At the end of the build end up with a big project
